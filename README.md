# Rock-Paper-Scissors-js-console-game-0.1
Here is the Planning:

La maquina elejira una de las opciones de manera aleatoria.

El usuario elejira una de las opciones.

Se comparan ambas elecciones.

Se elige un ganador dependiendo de algunos criterios.

Se avisa mediante un mensaje el ganador del match.

Hay un score que almacena el resultado de la partida para ambos jugadores.

Quien llegue a 5 victorias primero gana.

Se avisa mediante un mensaje el ganador de la partida.


Here is the Pseudocode:

La maquina tomara, de manera aleatoria, uno de los valores (Piedra, Papel o Tijeras), el usuario no debe saber cual es el resultado elejido por la maquina.

El usuario debera elejir uno de los valores (Piedra, Papel o Tijeras).

Se debera comparar ambos resultados y dependiendo de algunos factores determinar quien gana o quiern pierde:

    * Si ambos elijen la misma opcion, es empate.
    * Si uno elije Piedra y el Otro Papel, Quien elije Papel Gana.
    * Si uno elije Papel y otro elije Tijeras, Quien elije Tijeras Gana.
    * Si uno elije Piedra y el otro Tijeras, Quien elije piedra Gana.

Se debera mostrar el resultado dependiendo del ganador.

Debera de haber un marcador, que guarde el score de los jugadores.

El primero, de entre la maquimna o el usuario, que llegue a 5 ganadas, se acaba la partida y sale un mensaje proclamando al ganador definitivo
